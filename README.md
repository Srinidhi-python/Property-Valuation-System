# 🏠 Property Valuation System

A Machine Learning-based web application that predicts house prices based on property details such as location, square feet, BHK, and number of bathrooms. The application is developed using Python and Streamlit with a Random Forest Regression model.

## 🚀 Features

- 🔐 User Login Authentication
- 💰 House Price Prediction
- 📊 Analytics Dashboard
- 🗂️ Dataset Preview
- 🗺️ Property Location Map
- 🤖 Chatbot Assistant
- 📈 Prediction Confidence Meter
- 📄 Downloadable Property Report
- ✨ Modern User Interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- Joblib
- Git & GitHub

## 🤖 Machine Learning Model

- **Algorithm:** Random Forest Regressor
- **Prediction Accuracy:** Approximately 88%

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Srinidhi-python/Property-Valuation-System.git
```

2. Navigate to the project folder:

```bash
cd Property-Valuation-System
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```

5. Open the URL displayed in the terminal (usually `http://localhost:8501`).
